# **App Name**: Curiosity Engine

## Core Features:

- Interest Profiler: Collect user interests (movies, events, hobbies) as the basis for learning journeys.
- Daily Topic Generator: Use the OpenWebText Corpus and Wikipedia Articles Dataset as tools to select relevant daily topics based on user interests.
- Reading Material Curator: Find readings, articles, and resources that support the daily learning topic.
- Micro-Quiz Builder: Generate daily micro-quizzes to test understanding of the daily topic. Serve quizzes based on scraped websites and/or APIs.
- Learning Journey Tracker: Track user progress through learning journeys, noting completed topics and quiz scores.
- Curiosity Points: Award curiosity points for completing daily topics and quizzes.
- Leaderboard Display: Show the user's ranked progress against other users with a gamified leaderboard.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to evoke curiosity and intelligence.
- Background color: Light gray (#F5F5F5) for a clean, focused learning environment.
- Accent color: Teal (#009688) to highlight important information and calls to action.
- Body and headline font: 'PT Sans' for a modern look.
- Use simple, clear icons to represent different learning categories and topics.
- Clean and intuitive layout that allows users to easily navigate topics, readings, and quizzes.
- Subtle animations to provide feedback on user interactions.
